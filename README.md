# 💧 H2OyVida AI Sales Bot

Chatbot inteligente desarrollado en Python y Telegram para automatizar la atención al cliente, asesoría sobre ahorro de agua y generación de oportunidades comerciales para **H2OyVida**.

---

## 🚀 Descripción

Este proyecto implementa un asistente conversacional basado en IA capaz de:

- Asesorar clientes sobre ahorro de agua
- Explicar soluciones Aqua Saver
- Detectar interés comercial
- Guiar conversaciones hacia oportunidades de venta
- Automatizar el primer contacto con clientes

El bot funciona como un **asesor digital 24/7** para empresas y hogares interesados en eficiencia hídrica.

---

## 🧠 Tecnologías utilizadas

- Python 3
- Telegram Bot API
- OpenRouter AI
- HTTPX / Requests
- python-telegram-bot
- dotenv
